The aim of the project was to create a reimagination of NIIT University's Website in terms of it's design and functionality

• The application was developed using Angular, HTML & CSS.
• The website has been designed as a single page application where the jumbotron home page remains constant whereas all other content on the website changes with respect to the user's inputs.
